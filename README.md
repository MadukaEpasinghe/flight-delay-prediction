# ✈️ Flight Delay Prediction

> 🎯 Predicting flight delays using pre-departure data to improve operational efficiency and planning

---

## 📌 Overview
This project builds a machine learning model to predict whether a flight will be delayed based on pre-departure information such as weather, scheduling, and operational factors.

---

## 🧠 Business Problem
Flight delays cause significant disruptions, financial losses, and poor customer experience.  
Airlines need early predictions to take proactive measures.

---

## 💡 Solution
A predictive model was developed to classify flights as:
- On-time
- Delayed

This enables better scheduling, resource allocation, and customer communication.

---

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Data Visualization

---

## 📊 Features Used
- Departure time
- Weather conditions
- Airline/operator
- Airport data

---

## 📈 Model Performance

- Accuracy: 60%
- Precision: 60%
- Recall: 59%

---

## 🔍 Key Insights
- Weather conditions significantly impact delays
- Certain time slots have higher delay probability
- Airline-specific patterns can influence delay rates
